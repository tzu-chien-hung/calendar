Bootstrap Select 下拉框

基本select插件效果

爲下拉框添加插件支持 ， class="selectpicker"

單選模式示例

class="selectpicker show-tick" ， 選中項內容後添加" √ "小圖標

title="請選擇一項" ， 未設置默認選項並未選擇時 ， 組件框中的提示內容

data-live-search="true" ， 爲組件添加查詢功能

data-size="5" ， 設置下拉長度

多選模式示例

title="請選擇一項" ， 未設置默認選項並未選擇時 ， 組件框中的提示內容

data-live-search="true" ， 爲組件添加查詢功能

data-size="5" ， 設置下拉長度

multiple ， 添加可多選支持

data-max-options="2" ， 設置最多可選個數

該屬性在設置data-max-options="2"後 ， 建議不使用data-actions-box="true" ， 該屬性是添加"選擇所有"和"全部取消選擇"按鈕

分組選項示例

使用optgroup標籤將選項分爲一組， 下面介紹optgroup標籤屬性

label="分組名" ， 爲分組添加名稱

data-subtext="追加內容" ， 爲分組追加名稱

選項標籤中屬性介紹示例1

title="代稱" ， 改變選項在組件框中顯示內容， 相當於選項的代稱

selected ， 默認選中

data-icon="圖標代碼"爲選項前面添加小圖標， 如果沒設置 title="代稱"屬性， 那麼小圖標隨選項內容一起顯示在組件框中

data-subtext="提示文字" ， 在選項內容後添加提示(淡灰色小2號字體)文字

選項標籤中屬性介紹示例2

data-content="html代碼" ， 使用該屬性中的代碼片段替換該選項的內容， 以及選中後在組件框中顯示的也是該html代碼

如果感覺顯示在組件代碼框中的內容不好 ， 使用title="代稱" ， 改變選項在組件框中顯示內容， 相當於選項的代稱

其他屬性介紹

disabled ， 該屬性設置在組件select標籤中爲禁用組件 ; 設置在分組標籤(optgroup)中禁選該組 ; 設置在選項標籤(option)中禁選該項。

data-divider="true" ， 在選項與下一選項中間添加分割線。

Core options(核心選項)

核心選項也就是一些屬性，重要的屬性已經在上面介紹完了

寫法：data-選項名稱:對應值，組合成屬性

核心方法

$('#selectID').selectpicker('selectAll'); // 當爲多選下拉框時 ，執行全部選中

$('#selectID').selectpicker('deselectAll'); // 當爲多選下拉框時 ，執行全部取消選中

$('#selectID').selectpicker('refresh'); // 當你刪除或者添加選項時 ，刷新代碼

$('#selectID').selectpicker('hide'); // 隱藏下拉框

$('#selectID').selectpicker('show'); // 顯示下拉框

核心事件

$('#selectID').on('show.bs.select', function (e) {}); // 當點擊下拉框時，選項項窗體即將顯示觸發

$('#selectID').on('shown.bs.select', function (e) {}); // 當點擊下拉框時，選項項窗體顯示觸發

$('#selectID').on('hide.bs.select', function (e) {}); // 當點擊下拉框時，選項項窗體即將隱藏觸發

$('#selectID').on('hidden.bs.select', function (e) {}); // 當點擊下拉框時，選項項窗體隱藏觸發

$('#selectID').on('loaded.bs.select', function (e) {}); // 當下拉框被初始化後觸發

$('#selectID').on('refreshed.bs.select', function (e) {}); // 當刷新方法(也就是執行該方法.selectpicker('refresh'))下拉框時觸發

$('#selectID').on('changed.bs.select', function (e) {}); // 當值被改變時觸發
